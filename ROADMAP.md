# Roadmap

Este roadmap orienta o futuro do AJ. Ele deve ser atualizado quando prioridades mudarem ou quando uma entrega relevante for concluida.

## v0.8

- [x] Dashboard operacional
- [x] Perfil do usuario
- [x] Hub de monitoramento
- [x] Monitoramento sistemico
- [x] Monitoramento SEI
- [x] Reordenacao de prioridades
- [x] Design system inicial
- [x] Governanca para agentes com `AGENTS.md`
- [x] ADRs iniciais
- [x] Prompts reutilizaveis
- [x] Esteira de review com Backend, Reviewer, QA e Docs
- [x] Memoria do projeto em `.ai/context/`

## v0.9

- [ ] API interna para consultas e integracoes
- [ ] Alertas para demandas antigas, urgentes ou sem movimentacao
- [ ] Timeline de eventos por demanda
- [ ] Analytics operacional com comparativos por periodo
- [ ] Melhorias nos filtros do monitoramento
- [ ] Exportacoes adicionais alem de CSV
- [ ] Services dedicados para consolidacoes mais complexas
- [ ] Automatizar execucao real da esteira de review em CI ou script local

## v1.0

- [ ] Auditoria de acoes relevantes
- [ ] Logs estruturados de operacao
- [ ] Permissoes por perfil e area
- [ ] Rotinas mais robustas de importacao SEI
- [ ] Camada de inteligencia assistiva para analise e priorizacao
- [ ] Documentacao operacional completa para usuarios internos
